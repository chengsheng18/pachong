# dht-spider

----

本项目已由Golang重写，并迁移至[neoql/btlet](https://github.com/neoql/btlet)

新项目提升：

1. 更高地爬取效率，每分钟爬取资源数是以前的十倍以上
2. 除了爬取磁力链接外还获取资源信息
3. 更合理的API设计，方便扩展

----

**How to use**

```
python3 launch.py
```
